## 🎨 COLOR PALETTE

### Primary Colors
- **Amber 400-600**: Main brand color
- **Orange 500-700**: Accent and gradients
- **Gray 50-900**: Neutral backgrounds

### Gradients Used
- `from-amber-500 to-orange-600` - Buttons, CTAs
- `from-amber-400 to-orange-600` - Icons, accents
- `from-black/80 via-black/60 to-black/40` - Hero overlays
- `from-amber-100 via-orange-50 to-amber-100` - Section backgrounds

## 📐 SPACING & LAYOUT

### Section Spacing
- **py-24**: Standard section padding (increased from py-16)
- **mb-16**: Section title margin
- **gap-8**: Grid gaps for cards

### Typography Scale
- **Headings**: 4xl-8xl (Playfair Display)
- **Body**: base-xl (Inter)
- **Subtitles**: lg-2xl

---

## 🎭 ANIMATIONS & TRANSITIONS

### Hover Effects
- **Cards**: -translate-y-2 (lift effect)
- **Images**: scale-110 (zoom)
- **Icons**: rotate-6 (playful tilt)
- **Buttons**: -translate-y-1 + shadow-lg

### Transition Durations
- **Fast**: 300ms (buttons, links)
- **Medium**: 500ms (cards)
- **Slow**: 700ms (images)

## 🌟 GLASSMORPHISM IMPLEMENTATION

.glass-card {
  background: white/80 (dark: gray-800/80)
  backdrop-blur: xl
  border: white/20 (dark: gray-700/50)

  shadow: xl

Used in:

- Store cards
